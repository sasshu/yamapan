# yamapan
## アプリ概要
このアプリは「ヤマザキ春のパン祭り」でお皿をもらうために、あとどれだけシールを集めればよいかを計算してくれます。

This app enable us to calculate points of seals easily for bread festival in spring held by YAMAZAKI BAKING COMPANY.
